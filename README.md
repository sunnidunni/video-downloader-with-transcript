# video-downloader-with-transcript
This program automatically scrapes and downloads the mp4 after being provided a url, and automatically transcripts the video.

If you open the program on google colab, simply provide a url (ex. youtube, instagram, linkedin) and it will return an mp4 download through your browser, as well as output the transcript of the video. 

colab link: https://colab.research.google.com/drive/14q-q10T9q_aXGjRA1o3glyJhz_e_ZRvB?usp=sharing#scrollTo=M1UARqrb7ysm

This program uses yt-dlp and openai's whisper.
